# musicrename

[![Build Status](https://travis-ci.org/mfinelli/musicrename.svg?branch=master)](https://travis-ci.org/mfinelli/musicrename)

Assumes the working directory is in the format:

```
Artist 1 >
    [YYYY] Album 1 >
        artwork >
            scan 01.png
            scan 02.png
        01 Track 1.flac
        02 Track 2.flac
        folder.jpg
    [YYYY] Album 2 >
        artwork >
            scan 01.png
            scan 02.png
        01 Track 1.flac
        02 Track 2.flac
        folder.jpg
Artist 2 >
    ...
```
